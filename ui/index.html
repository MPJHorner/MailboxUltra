    Waiting for mail

    Send something to the SMTP port — any client, any message:

    swaks --to dev@example.com --from app@example.com \
  --server 127.0.0.1:1025 \
  --header "Subject: Hello from MailBoxUltra" \
  --body "It works."

    Keyboard shortcuts

    j / ↓Next message
    k / ↑Previous message
    gJump to newest
    GJump to oldest
    /Focus search
    1–6Switch tabs
    pPause / resume capture display
    dDelete current message
    Shift + XClear all
    tToggle theme
    ?Show this help
    EscClose dialog / blur search

    Modifier keys (⌘ / Ctrl / Alt) are never intercepted, so ⌘+C still copies and ⌘+R still reloads.

    Upstream relay

    Capture each message and then forward it to a real upstream MTA. Useful when you want to inspect mail and still deliver it. The host part of the URL is the SMTP server; smtp:// sends in clear, smtps:// wraps in TLS.
